# scala-raytracing
Raytracing with scala and processing

scala + procesingでレイトレーシングするサンプル

CPU並列化してあるのでコア数が多ければ高速に描画されます

## Usage
IntelliJ IDEAでプロジェクトを開いて、RayTracingまたはPathTracingオブジェクトを直接実行する
